8 bit Carry Look-Ahead Adder:

Carry Out was directly written bit-by-bit in the terms of given input, after which Sum was directly calculated using P and Carry Out.
Used a for loop and a system function $urandom_range to produce the inputs in the test bench file.

Johnson Counter:

Always block was used to check for positive edge of clock and reset as well as to accommodate if-else conditions. A single statement was used to derive the next state from the previous state.
Used a for loop and a system function $urandom_range to produce the inputs in the test bench file.
